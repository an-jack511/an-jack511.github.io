## 遗传与进化

1. 在杂交育种中，雄性不育植株的优点是无需进行 **去雄**~~（+套袋）~~，大大减轻了杂交操作的工作量（统练 1-17-2507）
   
2. ENU处理通常会造成DNA的碱基替换，引发基因组随机突变。从基因和蛋白质的关系分析，编码序列的改变可能导致的效应包括:个别氨基酸的改变导致蛋白质结构变化;终止密码子出现的位置改变，导致蛋白质截短或变长;以及因 **密码子的简并性**，氨基酸序列不改变。
   
    > 从密码子表可以看出,像苯丙氨酸、亮氨酸这样,绝大多数氨基酸都有几个密码子,这一现象称作密码子的 *简并* ——B2 P67
    > ![20260123193400](https://raw.githubusercontent.com/an-jack511/blogIMG/main/MyBlogImg20260123193400.png)
    > 可以从增强密码子容错性的角度来解释，当密码子中有一个碱基改变时，由于密码子的简并性，可能并不会改变其对应的氨基酸；也可以从密码子的使用频率来考虑；当某种氨基酸使用频率高时；几种不同的密码子都编码同一种氨基酸可以保证翻译的速度。——教师用书 B2 P216

    > **基因通过控制酶的合成来控制代谢过程，进而控制生物体的性状。**<br>
    > **基因还能通过控制蛋白质的结构直接控制生物体的性状。** ——B2 P71-72

6. 多种植物病原体独立进化，虽亲缘关系很远但均有与 C4 基因相似的基因。这是不同病原体在同一选择压力（如 SADR）下 **进化的方向相同（趋同进化）** 的结果。（**期末** -17-2601）

    > 趋同进化是指不同物种在相似的环境中发育出相似的性状的进化过程，它是不同生物在相同环境条件下得到相同选择的结果。例如，空中的鸟类与蝙蝠，海中的鲨鱼与海豚，它们在外形上都表现出明显的相似，但在亲缘关系上却分别隶属于不同的纲。又如，非洲的大戟属植物与美洲的多种仙人掌植物虽然是截然不同的种子植物，但外形却很相似。——教师用书 B2 P324

## 生物环境

1. 生态修复领域的重要结论“梦境假说”认为，停止干扰后，食物链就会从低营养级到高营养级逐步恢复，群落最终会达到一个 **与群落所处环境相适应** ~~（物种组成）~~ 的相对稳定的状态。（统练 7-17-2511）
   
   > 影响群落演替的因素，有群落外界环境的变化，生物的迁入、迁出，群落内部种群相互关系的发展变化，以及人类的活动，等等。这些因素常常处于变化的过程中，**适应变化的种群数量增长或得以维持，不适应的数量减少甚至被淘汰**，因此，群落就不断地演替。无论裸岩上的群落演替，还是弃耕农田上的演替，最终都会达到一个与群落所处环境相适应的相对稳定的状态。（注：即顶极群落）——XB2 P41


2. 生态系统中能量的 **输入**~~（同化）~~、传递、转化和<散失>的过程，称为生态系统的能量流动。（统练 8-16-2601）
   
   > **生态系统中能量的输入、传递、转化和散失的过程，称为生态系统的能量流动。** ——XB2 P54

3. 
| 概念组 | 术语名称 | 简写/公式 | 考试定义的内涵与应用 | 易错点拨 |
| :--- | :--- | :--- | :--- | :--- |
| **输入端** | **摄入量** | $Input$ | 动物吞进嘴里的总能量。 | 包含“未同化”的粪便能。 |
| | **同化量** | **A** (Assimilation) | 真正进入生物体并被吸收转化的能量。 | **A = 摄入量 - 粪便量**。是该级的总能量起点。 |
| **输出端** | **呼吸量** | **R** (Respiration) | 以**热能**形式散失的不可逆能量。 | 能量流动中占比最大的去路，不能被再利用。 |
| | **生发繁** | **P / NPP** | **用于生长、发育和繁殖的能量**。 | **P = A - R**。储存在活体有机物中的能量。 |
| **去路端** | **流向下一级** | - | 被高营养级生物捕食并同化的部分。 | 计算**传递效率 (10%~20%)** 的分子。 |
| | **流向分解者** | **D** (Decomposer) | 遗体残骸、**凋落物**、下级粪便。 | 只要是“脱离活体”的有机物能量都归此类。 |

| **状态/速率** | **生物量** | **Biomass** | 单位面积内生物有机体的**总干重**。 | 属于“存量”，**不能直接**替代能量算传递效率。 |
| | **总初级生产量** | **GPP** | 生产者固定的太阳能总量。 | 等于第一营养级的**同化量**。 |
